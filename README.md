# OCRapp
## 使い方
* 動作環境
Mac OS:10.14.1  
Python:3.6.7  
下記を全てインストールしてください。  
PyQt5:5.11.3  
scipy:1.1.0  
tensorflow:1.12.0  
cloneして以下を実行
`$ cd https://github.com/zakopuro/OCRapp.git`  
`$ cd OCRapp`  
`$ python main.py`
